Detailed Description

template<int n, int m, class ConType>
class altro::constraints::ConstraintValues< n, m, ConType >

A constraint that also allocates memory for constraint values, Jacobians, dual variables, etc.

This class also provides methods for evaluating terms such as the augmented Lagrangian.

Template Parameters
nCompile-time state dimension
mCompile-time control dimension
ConTypeType of constraint (equality, inequality, conic, etc.)

Evaluate the augmented Lagrangian.

The augmented Lagrangian for an optimization problem of the form:

\begin{aligned} \text{minimize} &&& f(x) \\ \text{subject to} &&& c(x) \in K \\ \end{aligned}

is defined to be

\[ f(x) + \frac{1}{2 \rho} (||\Pi_{K^*}(\lambda - \rho c(x))||_2^2 - ||\lambda||_2^2) \]

where \( \lambda \) are the Lagrange multipliers (dual variables), \( \rho \) is a scalar penalty parameter, and \( \Pi_{K^*} \) is the projection operator for the dual cone \( K^* \).

Parameters
xState vector
uControl vector
Returns
The augmented Lagrangian for the current knot point, evaluated at x and u.

◆ UpdateDuals()

template<int n, int m, class ConType >
void altro::constraints::ConstraintValues< n, m, ConType >::UpdateDuals ( )
inline

Update the dual variables.

Updates the dual variables using the current constraint and penalty values. The resulting dual variables are projected back into the dual cone such that they are always guaranteed to be feasible with respect to the dual cone.

The update is of the form:

\[ \lambda^+ - \Pi_{K^*}(\lambda - \rho c) \]
