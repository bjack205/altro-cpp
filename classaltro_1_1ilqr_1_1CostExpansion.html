Detailed Description

template<int n, int m>
class altro::ilqr::CostExpansion< n, m >

Stores the first and second-order expansion of the cost The underlying memory can either be stored on the heap or the stack, depending on whether the state and control dimensions are specified at compile time via the type parameters. Use Eigen::Dynamic to allocate on the heap.

Template Parameters
ncompile-time state dimension
mcompile-time control dimension
