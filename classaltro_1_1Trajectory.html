Detailed Description

template<int n, int m, class T = double>
class altro::Trajectory< n, m, T >

Represents a state and control trajectory.

If the number of states or controls across the trajectory is not constant the associated type parameter can be set to Eigen::Dynamic.

Template Parameters
nsize of the state vector. Can be Eigen::Dynamic.
msize of the control vector. Can be Eigen::Dynamic.
Tfloating precision of state and control vectors
