Detailed Description

template<int n, int m, class T = double>
class altro::KnotPoint< n, m, T >

Stores the state, control, time, and time step for a single knot point.

The state and control vectors can live on either the stack or the heap, leveraging Eigen's Matrix class. If

Template Parameters
nor
mare equal to Eigen::Dynamic, the vector will allocated on the heap.

Use StateDimension and ControlDimension to query the actual state or control dimension. Use StateMemorySize and ControlMemorySize to get the type parameters.

Template Parameters
nsize of state vector. Can be Eigen::Dynamic.
msize of state vector. Can be Eigen::Dynamic.
Tprecision of state and control variables
