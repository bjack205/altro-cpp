Detailed Description

template<int n, int m>
class altro::StateControlSized< n, m >

A super class that stores the state and control dimension.

With an option to store them at compile time for statically-sized data structures.

Here's an example of basic inheritance

template <int n, int m>
class Derived : public StateControlSized<n,m> {
public:
Derived(int state_dim, int control_dim)
: StateControlSized<n,m>(state_dim, control_dim) {}
}
