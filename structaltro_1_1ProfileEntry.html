Detailed Description

An entry in the profiler summary table.

Records the hierarchical structure of the profiling results. When parsed correctly, these should be arranged in a tree-like structure that is used to calculate the total time and time relative to the parents.

This is used internally by the Timer class to print the results via PrintSummary.
