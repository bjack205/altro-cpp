Detailed Description

Basic threadpool that uses a single work queue.

The threadpool is created via LaunchThreads(nthreads), which creates nthreads "spinning" threads. Tasks can be added to the work queue via AddTask, which accepts anything that can be converted to a std::packaged_task<void()>. The user can wait for all the submitted jobs to finish via Wait.

The pool is not copyable, but is moveable.

Usage

ThreadPool pool;
# Create the tasks
pool.AddTask(std::packaged_task<void()>(...));
# Launch the threads once all tasks have been created
pool.LaunchThreads();
# Wait for the threads to finish
pool.Wait();
# Close the thread pool
pool.StopThreads();

Based off of Listing 9.1 in C++ Concurrency in Action by Anthony Williams.
