Detailed Description

template<class Model, class Integrator = RungeKutta4<Model::NStates, Model::NControls>>
class altro::problem::DiscretizedModel< Model, Integrator >

Discretizes a continuous dynamics model using an explicit integrator.

Uses a specified integrator to integrate a continuous time dynamics model over a discrete time step.

Template Parameters
ModelModel to be discretized. Must inherit from FunctionBase.
IntegratorAn explicit integrator. Should inherit from ExplicitIntegrator.

For best performance, Model::NStates and Model::NControls should provide compile-time information about the number of states and controls. This will allow the integrator to allocate memory on the stack for any temporary arrays needed during the integration procedure.
